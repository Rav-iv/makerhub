# how to get help
- keep a notes.md file next to a js script
- comment code in the script to understand what you are doing
- isolate the problem to a single line
- simplify the problem to a single line of node with defined input and output expected
- write out the simplified problem in a markdown file 
- search google (the answer will probebly be in stack overflow)
- keep a record of different solutions tried and why they didn't work
- write to stack over flow
- write to reddit
- ask me
